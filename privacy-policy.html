FAQCheck Privacy Policy

Last updated: July 26, 2025

Overview

FAQCheck is committed to protecting your privacy. This Chrome extension does not collect, store, or transmit any personal information.

What Information We Store

FAQCheck stores the following information locally on your device:

  • API Key: Your Claude API key is stored securely in Chrome's encrypted storage system
  • Usage Statistics: Basic counters of how many fact-checks you've performed (stored locally only)
  • History: The first 100 characters of texts you've fact-checked (stored locally only, never transmitted)

What Information We Send

FAQCheck sends the following information:

  • Selected Text: When you fact-check text, it is sent to Claude AI (Anthropic) for analysis
  • API Key: Your API key is sent to Anthropic to authenticate requests

What We Don't Do

  • We don't track your browsing history
  • We don't collect personal information
  • We don't share data with third parties (except Anthropic for AI processing)
  • We don't store your queries on any server
  • We don't use analytics or tracking tools

Third-Party Services

FAQCheck uses the Claude AI API by Anthropic for fact-checking analysis. When you use the extension, your selected text is sent to Anthropic's servers for processing. Please review Anthropic's Privacy Policy for information about how they handle data.

Data Security

Your API key is stored using Chrome's secure storage API, which encrypts sensitive data. All communication with the Claude AI API happens over secure HTTPS connections.

Data Deletion

You can delete all stored data at any time by:

  • Clicking "Clear History" in the extension popup
  • Uninstalling the extension
  • Using Chrome's clear browsing data feature

Changes to This Policy

We may update this privacy policy from time to time. We will notify users of any material changes by updating the "last updated" date.

Contact

If you have questions about this privacy policy or the extension, please create an issue on our GitHub repository.

Consent

By using FAQCheck, you consent to this privacy policy.
